project berupa sebuah web yang dibagi menjadi 2 bagian yaitu sisi user mahasiswa dan juga sisi admin

1. sisi user admin
dalam halaman admin pengguna diwajibkan login untuk masuk kedalam sistem. setelah login admin sampai di halaman dashboard yang menampilkan data - data. 
lalu di sebelah kiri akan ada side bar yang memiliki beberapa menu sebagai navigasi untuk admin. akan ada keterangan username admin dan juga keterangan admin yang masuk
lalu di bawahnya akan ada menu dashboard
dibawah dashboard akan ada menu dapodik yang menampilkan data - data mahasiswa yang terdaftar. dapodik ini bisa diedit dan juga dihapus. 
dibawah dapodik akan ada menu pengumuman yang bisa diakses oleh admin. pengumuman ini memiliki fitur untuk menambah pengumuman berupa judul, deskripsi, dan juga dokumen pengumuman. pengumuman juga bisa diedit maupun dihapus. 

2. sebelum masuk ke sisi user. mahasiswa diwajibkan mendaftar terlebih dahulu dan mengisi biodata. lalu setelah mahasiswa mendaftar ia akan mendapatkan nim dan password yang akan tergenerate secara otomatis. yang bisa melihat nim dan pass hanya admnistartor dalam hal ini adalah admin. (dalam kondisi nyata harus melalui pelunasan keuangan baru bisa mendapatkan password website)

2. sisi user mahasiswa
dalam sisi mahasiswa diwajibkan login terlebih dahulu dengan nim dan pass yang diberikan. lalu setelah mahasiswa login akan masuk ke halaman dashbord. di halaman dashboard akan ada data mahasiswa yang sudah diisi oleh mahasiswa sebelumya
lalu akan ada sidebar yang menjadi navigasi untuk mahasiswa. di sidebar terdapat data nim dan nama mahasiswa yang login yang ditampilkan. 
dibawah dashboard akan ada menu untuk perkuliahan. disana akan ada menu untuk melihat jadwal. namun sebelum jadwal dibuat mahasiswa harus input krs. 
input krs dibuat sesuai dengan semester mahasiswa. 
setelah mahasiswa mengisi krs akan di return kembali ke halaman perkuliahan untuk menampilkan jadwal kuliah. 
lalu dibawah menu perkuliahan akan ada menu pengumuman. disini mahasiswa dapat melihat pengumuman terbaru yang dibuat oleh admin dan bisa mendownload file yang tersedia.
